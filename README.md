# chuanzhi_easy
这是跟着黑马程序员做的一个简单版本的传智书城的网页。
技术包含：简单的控件布局：css+div，以及简单的JavaScript验证表单提交的数据是否符合规范。
正则表达式，获取表单的value，验证表单数据规范性。
布局主要是：分为5个部分，导航菜单栏，商品分类，商品详细信息，版权信息。
